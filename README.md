# Harmonizadora Elite - Landing Page

Landing page/hub responsiva para a marca **Harmonizadora Elite**, criada com React + Vite + Tailwind CSS.

## 🚀 Como instalar

```bash
npm install
```

## 💻 Como rodar localmente

```bash
npm run dev
```

O projeto estará disponível em `http://localhost:5173`.

## 📦 Como gerar build

```bash
npm run build
```

Os arquivos estáticos serão gerados na pasta `dist/`, prontos para deploy.

## 🗂️ Estrutura do projeto

```
src/
  ├── components/
  │   ├── ProfileHeader.jsx   # Cabeçalho com foto e título
  │   ├── CtaBlock.jsx        # Bloco individual de CTA
  │   └── Footer.jsx          # Rodapé com links legais
  ├── data/
  │   └── links.js            # Dados centralizados (CTAs, links, config)
  ├── App.jsx                 # Componente principal
  ├── main.jsx                # Entry point
  └── index.css               # Estilos globais + Tailwind
```

## 🎨 Stack

- **React 18**
- **Vite 5**
- **Tailwind CSS 3**
- **Sem backend** — site 100% estático
